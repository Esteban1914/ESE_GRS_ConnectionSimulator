# Proyecto ESE_GRS_ConnectionSimulator v4.5

Este proyecto tiene como objetivo el desarrollo de una aplicación
que permita la simulación entre el brazo ESE_GRS y la 
[ESE_GRS_App](https://github.com/Esteban1914/ESE_GRS-Class)
.Permite simular desde el Puerto Serie, desde un "Puente" entre 
el COM y un ServerSocket local; el cual se conecta y comunica 
correctametne con el Server de la Web, además permite conectarse como 
el cliente ESE en la Web.
